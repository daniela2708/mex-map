# Logos

This folder contains brand logos used in the application.

## Current Logos

- `wizeline.svg` - Wizeline company logo (used in header)
- `pepsi.svg` - Pepsi brand logo
- `coca-cola.svg` - Coca-Cola brand logo (currently empty - needs manual download)

## How to Update Coca-Cola Logo

The Coca-Cola logo file is currently empty due to download restrictions. To add it:

1. Download the logo manually from: https://www.coca-colacompany.com/
2. Or use an alternative source for the Coca-Cola SVG logo
3. Save it as `coca-cola.svg` in this folder
4. Ensure it's a valid SVG file
